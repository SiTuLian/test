创建canvas

步骤1.创建画布，代码如下

		<canvas id="canvas">您的浏览器不支持canvas，请换一个！</canvas>

步骤2.对画布进行宽高的设置，我们可以用如下方法进行设置

第一种利用Javascript设置

		<script type="text/javascript">
			var canvas = document.getElementById('canvas');//获取画布的ID

			//设置画布的宽度和高度
			canvas.width = 500;
			canvas.height = 500;

			//以上同样设置了画布的宽高为500px;
		</script>

第二种利用Javascript获取屏幕的宽高

		<script type="text/javascript">
			var canvas = document.getElementById('canvas');//获取画布的ID

			//设置画布的宽度和高度
			canvas.width = window.innerWidth;
			canvas.height = window.innerHeight;

		
		</script>

第三种直接在行内样式设置

		<canvas id="canvas" style="width: 500px; height: 500px;">您的浏览器不支持canvas，请换一个！</canvas>

步骤3.此时画布的宽高已经设置好了，但是并没有看见，我们可以给它添加样式，比如背景颜色或边框。

		<style type="text/css">
			/*添加背景颜色为红色*/
			#canvas{
				background-color: red;
			}
			/*添加边框*/
			#canvas{
				border: 1px solid #333333;
			}
		</style>

注意！

如果浏览器不支持canvas，则不会显示画布，而会输出“您的浏览器不支持canvas，请换一个！”

不要在CSS样式表里直接定义canvas的宽高，通过CSS样式控制与直接在Canvas标签内定义width（宽）和height（高），他们在浏览器上显示的效果是不一样的。通过CSS样式控制的canvas里面的内容会被自动放大了。而在Canvas标签内容定义的width和height就不会。

Canvas标签包涵两个东西，一个是元素本身，另一个是元素绘图表面（drawing surface）。他们都有着自己各自的尺寸大小。所在canvas的大小实际上是指两方面的，元素本身的大小和元素绘图表面大小。Canvas元素本身和元素绘图表面默认的大小都是300x150像素。

两个设置Canvas大小的方法的区别在于，通过CSS设置Canvas大小时，只对Canvas元素本身的大小进行设置，而没有对元素绘图表面大小进行设置，所以元素绘图表面是默认大小300x150像素。当Canvas元素本身大小与元素绘图表面不一样时，浏览器会对元素绘图表面进行缩放，使其符合元素大小。所以就会出现上面看到的两中不同的效果。而在canvas标签内直接定义width和height就不一样了，它是同时定义元素本身大小和元素绘图表面大小。通过JS来定义与在canvas标签内定义效果是一样的。
